## Installation

### Requirements

- Linux with Python ≥ 3.6
- PyTorch ≥ 1.4.0
- [DGL with CUDA](https://www.dgl.ai/pages/start.html)
  - Install [RDKit](conda install -c conda-forge rdkit) with `conda install -c conda-forge rdkit`.
- `pip install -r requirements.txt`
