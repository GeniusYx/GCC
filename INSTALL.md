## Installation

### Requirements

- Linux or macOS with Python ≥ 3.6
- PyTorch ≥ 1.3.1
- [PyG with CUDA](https://github.com/rusty1s/pytorch_geometric/#installation)
- [DGL with CUDA](https://www.dgl.ai/pages/start.html)

### Install CogDL

```bash
git submodule init
git submodule update
cd cogdl
pip install -e .
```
